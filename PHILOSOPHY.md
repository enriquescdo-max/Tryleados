## The Five Immutable Principles

### 1. Root-Node First
Solve the one problem that unlocks the entire branch. Before writing any
function, identify the ROOT NODE — the single bottleneck that, if removed,
unlocks an entire class of downstream possibilities.

### 2. Learning Systems over Expert Systems (AlphaZero Logic)
Move beyond if/then logic. Every agentic loop should learn from outcomes,
discover "Move 37" — the solution a human would never try — and improve
with each cycle. NEVER be capped at human-level performance.

### 3. In-Silico Before Action
Before ANY production action fires: run through a synthetic simulation
layer, model outcomes across edge cases and failure modes, optimize for
the Move 37 path. Only then release to production execution.

### 4. Agentic Autonomy with Hard Guardrails
Every autonomous workflow MUST have: hard guardrails that cannot be
bypassed, human verification tokens for irreversible actions, audit logs
in Supabase agent_context on every execution, graceful degradation on
guardrail trip.

### 5. Human Flourishing as the Exit Condition
Every feature ships only if: it removes repetitive cognitive tasks, gives
intelligence the human couldn't access before, and moves the human closer
to high-level relationship work. Would a solo operator with this tool
outperform a 10-person team without it?
